# 💼 Job Finder

Projeto desenvolvido durante meus estudos em **Node.js**, com o objetivo de criar uma plataforma para divulgação e busca de vagas de emprego.

O projeto foi desenvolvido acompanhando aulas práticas, aplicando conceitos importantes do desenvolvimento web com Node.js, banco de dados e arquitetura MVC.

---

## 📚 Objetivo

O principal objetivo deste projeto é praticar conceitos de desenvolvimento Back-end, como:

* Criação de servidores com Node.js
* Utilização do Express.js
* Rotas e Controllers
* Template Engine (Handlebars)
* Integração com banco de dados
* Operações CRUD
* Organização de projetos utilizando o padrão MVC

---

## 🚀 Tecnologias utilizadas

* Node.js
* Express.js
* Express Handlebars
* Sequelize
* SQLite
* HTML5
* CSS3
* JavaScript

---

## ⚙️ Funcionalidades

* ✅ Cadastro de vagas
* ✅ Listagem de vagas disponíveis
* ✅ Visualização dos detalhes da vaga
* ✅ Pesquisa de vagas
* ✅ Persistência dos dados utilizando banco de dados

## ▶️ Como executar o projeto

### Clone o repositório

```bash
git clone https://github.com/LucasDuarteV/projeto-job-finder.git
```

### Entre na pasta

```bash
cd projeto-job-finder
```

### Instale as dependências

```bash
npm install
```

### Execute o projeto

```bash
npm start
```

ou, caso utilize o Nodemon:

```bash
npm run dev
```

O projeto ficará disponível em:

```
http://localhost:3000
```
